Criar um mundo com Blocos

Vamos criar jogos!

Hoje você vai aprender a programar usando blocos, como peças de LEGO 🧩

Com eles, podemos criar movimentos, regras e jogos incríveis!

O que é a Programação em Blocos?
  • Programar é dar ordens ao computador
  • Você encaixa blocos para criar ações
  • Os blocos são como instruções
  • Não precisa digitar código!
Elementos do Jogo 🎮
  • Cenário stage 🏞️: o fundo do jogo
  • Pode mudar durante o jogo
  • Tem sons 🎵 e aparência
  • Botão RUN 🚩
Elementos do Jogo 🎮
  • Atores sprites 🏃‍➡️: personagens do jogo
  • Podem andar, pular, falar
  • Têm posição:
    • X → lado a lado
    • Y → cima e baixo
  • Também têm:
    • Tamanho
    • Direção
