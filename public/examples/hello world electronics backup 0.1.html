OLÁ MUNDO DA ELETRÔNICA
O que é um Computador
Máquina capaz de:
  • Receber informações
  • Processar dados
  • Cumprir tarefas, mostrar resultados ou armazená-los
Analogia entre um Ser Humano e um Robô👍🦾
Interfaces Homem–Máquina
Comunicação entre pessoas e sistemas:
  • Hardware (Físico): componentes internos, dispositivos e periféricos
  • Software (Virtual): programas digitais e instruções
  • Interação por interfaces
O que é um Algoritmo
Conjunto finito e bem definido de passos ordenados para realizar uma tarefa ou resolver um problema.
Exemplo: Beber água
  1. Pegar o copo
  2. Encher com água
  3. Levar à boca
  4. Beber
    5. 👉 Isso é um algoritmo!
Características dos Algoritmos
  • Tem início, meio e fim
  • Segue uma ordem lógica
  • Cada passo deve ser claro e sem ambiguidade
  • Deve resolver um problema
  • Pode ter zero ou mais entradas, mas pelo menos um resultado final (ou saída) é esperado
Exemplos do dia a dia
  • Receita de bolo
  • Manual de montar brinquedo
  • Regras de um jogo
  • Passo a passo para escovar os dentes
E qual é a diferença entre um algoritmo e um software?
Entrada → Processamento → Saída
Modelo de funcionamento dos sistemas:
  • Entrada → sensores e comandos
  • Processamento → microcontrolador
  • Saída → respostas e ações
Em nossas aulas, usaremos o microcontrolador Arduino, mas existem muitos outros...
