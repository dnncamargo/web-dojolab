A programação é baseada no uso de Algoritmos
O que é um ALGORITMO?
Os algoritmos são uma sequência de instruções detalhadas passo a passo.
  • São escritos por programadores para que uma máquina execute uma tarefa, como se fosse uma receita.
  • A máquina deve seguir todas as tarefas exatamente como escrito.
  • Um software é um conjunto de diferentes algoritmos, cada um com seu propósito
  • Em todo sistema robótico, seguimos esse ciclo:
Entrada → Processamento → Saída
Estrutura Básica dos Algoritmos

Todo algoritmo segue uma lógica de pensamento organizada em três partes principais:


🟦 Sequência
As ações acontecem uma depois da outra.
Ex: acender a luz → ligar o ventilador → sentar.

🟧 Repetição
Um bloco de ações que se repete enquanto uma condição for verdadeira.
Ex: repetir 10 vezes → piscar o LED.

🟨 Condição (seleção)
O algoritmo decide o que fazer dependendo de uma situação.
Ex: se o botão for pressionado → mover o robô.
Características dos Algoritmos
  • Tem início, meio e fim
  • Segue uma ordem lógica
  • Cada passo deve ser claro e sem ambiguidade
  • Deve resolver um problema
Exemplo: Fazer um sanduíche
  1. Pegar pacote de pão fatiado, pote de manteiga de amendoim, pote de geléia de amora, faca cega, prato
  2. Colocar duas fatias de pão no prato
  3. Abrir o pote de manteiga de amendoim
  4. Espalhe uma camada generosa de manteiga de amendoim em um dos lados de uma das fatias de pão.
  5. Espalhe a geleia de amora na outra fatia de pão
  6. Una as duas fatias de pão com o recheio para dentro
  7. NHAC!! NHAMMY!! HUUMMM!! 😋
👉 Isso é um algoritmo!
E qual é a diferença entre um algoritmo e um software?
Entrada → Processamento → Saída
Vamos fazer uma analogia com o corpo humano
Nós também somos sistemas que percebem o ambiente, processam informações e respondem ao meio.

Captamos informações por meio dos sentidos — visão, audição, tato, olfato e paladar. Essas informações chegam ao cérebro, onde são processadas e guardadas na memória. Em seguida, agimos com músculos, fala e outros mecanismos que influenciam o ambiente.
O Sistema Humano
👁️👂
Entradas
Sentidos

🧠
Processamento
Cérebro
(decisões e memória)
💪🗣️
Saídas
Músculos e fala
(ações no ambiente)
O Sistema das Máquinas
A robótica busca imitar esse mesmo processo:
🎛️
Entradas
Sensores
(luz, som, distância...)
🔲
Processamento

(como o Arduino)
⚙️💡🔊
Saídas
Atuadores
(motores, LEDs, buzzer, etc.)
Arduino, Sensores e Atuadores
Componentes dos projetos robóticos:
  • Arduino: cérebro do sistema
  • Sensores: captam dados do ambiente
  • Atuadores: executam ações físicas
Em nossas aulas, usaremos o microcontrolador Arduino, mas existem muitos outros...
